# senatestacks.com

##### Repo for senatestacks.com

## Build the app

- Clone this repository
- Install required dependencies

  ```
  yarn install
  cd client
  yarn install
  ```

- To run local server

  ```
  $ yarn start
  ```

## Technologies Used

- [React](https://reactjs.org/)
- [Node](https://nodejs.org/)
- [Express](https://expressjs.com/)
